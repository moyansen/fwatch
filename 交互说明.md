# 智能手表交互说明

## 快速操作指南

### 开机流程
```
关机状态 → 长按5秒 → Logo页(2秒) → 表情页(已激活) / 二维码页(未激活)
```

### 关机流程
```
开机状态 → 长按5秒 → 待机屏(黑屏)
注意：背负正常/异常状态下不可关机
```

### 页面切换
```
短按按键 → 按顺序循环: 表情 → 信息 → 联系人 → 步数 → 二维码 → 背负
```

### SOS求救
```
连按3次 → 进入SOS页面 → 显示紧急联系信息

退出方式：
- 非佩戴状态：再连按3次退出
- 佩戴状态：家长通过云端远程解除
```

## 背负状态与页面行为

| 背负状态 | 页面显示 | 息屏行为 | 特殊处理 |
|----------|----------|----------|----------|
| 未检测 | 表情页 | 10秒后息屏 | 按键唤醒 |
| 未背负 | 表情页 | 不息屏 | - |
| 背负正常 | 表情页 | 不息屏 | 禁止关机 |
| 背负异常 | 背负页(30秒倒计时) | 不息屏 | 倒计时结束自动切表情页 |

## 充电状态影响

### 充电时
- ✅ 自动开机并显示充电页
- ❌ 物理按键锁定（无法开关机、切页、SOS）
- ❌ 背负状态切换锁定
- ✅ 可显示二维码页（未激活设备）

### 拔充电线后
- 背负状态重置为"未检测"
- 已激活 → 表情页
- 未激活 → 二维码页
- 如SOS处于锁定状态，强制解除

---

## 激活状态影响

| 场景 | 已激活 | 未激活 |
|------|--------|--------|
| 开机后 | 进入表情页 | 进入二维码页 |
| 页面切换 | 可切换所有页 | 强制锁定二维码页 |
| 充电时 | 显示充电页 | 显示充电页 |
| 待机唤醒 | 返回原页面 | 返回二维码页 |

## 控制面板说明

### 物理按键
- 模拟手表侧面主按钮
- 支持鼠标按下/释放操作
- 显示长按进度环（蓝色圆圈）

### 背负状态按钮（2×2网格）
| 图标 | 状态 | 颜色 |
|------|------|------|
| ? | 未检测 | 灰色 |
| ✕ | 未背负 | 灰色 |
| ✓ | 背负正常 | 绿色 |
| ✗ | 背负异常 | 红色 |

### 充电按钮
- 点击切换充电/非充电状态
- 充电时按钮高亮

### 激活状态按钮
- 仅关机时可切换
- 已激活：绿色勾选
- 未激活：红色叉号

### 云端解除SOS按钮
- 橙色背景突出显示
- 仅在SOS激活时有效
- 强制解除SOS锁定状态

## 页面指示器

屏幕底部6个圆点，对应页面：
```
● 表情页    ● 信息页    ● 联系人页
● 步数页    ● 二维码页  ● 背负页
```
- 蓝色高亮 = 当前页面
- 灰色 = 其他页面

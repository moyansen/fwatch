# FILA 智能手表交互原型 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
FILA 儿童智能手表高保真交互原型

### 1.2 产品定位
儿童/学生智能手表，主打安全定位、紧急求救、背负姿态监测、设备异常感知等功能。

### 1.3 目标用户
- 6-15 岁学生群体
- 家长（远程管理端）

---

## 2. 核心功能模块

### 2.1 电源管理
| 功能 | 说明 |
|------|------|
| 开机 | 长按物理按键 **3 秒**启动，显示 FILA 品牌 Logo 2 秒后进入首屏 |
| 关机 | 长按物理按键 **3 秒**关闭（佩戴中、SOS 状态下不可关机） |
| 待机 | 未检测 / 网络异常 / 设备离线状态下 10 秒无操作自动息屏 |
| 唤醒 | 短按按键从待机唤醒，若仍有未解除的异常则回到异常页 |

### 2.2 页面清单
| 页面 | 内容 | 说明 |
|------|------|------|
| 待机屏 | 黑屏 | 省电模式 |
| Logo 屏 | FILA 品牌 Logo + 加载动画 | 开机过渡页面 |
| 表情页 | 动态表情 😊 | 主界面/首页 |
| 基本信息页 | 姓名 / 性别 / 生日 / 身高 / 体重 | 标题"基本信息" |
| 紧急信息页 | 姓名 / 紧急电话 / 血型 / 过敏原 / 基础病 | 标题"紧急信息" |
| 二维码页 | 绑定二维码 + IMEI 号 | 标题"设备IMEI号" |
| 背负页 | 背负状态显示 | 姿势监测 |
| SOS 页 | 紧急联系人 / 紧急电话 / 血型 / 过敏原 / 基础病 | 红色警示界面 |
| 充电页 | 充电表情 🔋 | 充电中显示 |
| 异常页 | 网络异常 / 设备离线 / 电量过低 | 设备健康提示 |
| 升级页 | 固件升级进度条 | OTA 升级中显示 |

页面循环顺序（短按切换）：`表情 → 基本信息 → 紧急信息 → 二维码 → 背负 → 表情...`

### 2.3 背负检测系统
| 状态 | 图标 | 颜色 | 页面行为 |
|------|------|------|----------|
| 未检测 | ? | 灰色 | 10 秒息屏；背负页回显 `lastWornStatus` |
| 未背负 | ✕ | 灰色 | 进入 SOS 锁定（佩戴数据为 0 视为异常） |
| 背负正常 | ✓ | 绿色 | 直接显示表情页，不展示"背负正常"字样，顶部状态指示器隐藏 |
| 背负异常 | ✗ | 红色 | 显示背负页，**无 30 秒倒计时**，停留直到状态变化；具体子类型见 2.3.1 |

#### 2.3.1 背负异常子类型
| 子类型 | 说明 |
|----------|------|
| 左单肩背负 | 仅左肩受力，建议双肩均衡背负 |
| 右单肩背负 | 仅右肩受力，建议双肩均衡背负 |
| 左重右轻失衡 | 左侧重量明显较重，左右负重失衡 |
| 右重左轻失衡 | 右侧重量明显较重，左右负重失衡 |

背负页与顶部状态指示器均显示子类型的具体中文描述（如“左单肩背负”）；点击控制面板中任一子类型按钮即可自动进入背负异常状态。

### 2.4 SOS 紧急求救
- **触发方式**：连按物理按键 3 次
- **退出方式**：
  - 非佩戴状态：再次连按 3 次可退出
  - 佩戴状态（未背负 / 背负正常 / 背负异常）：仅限**云端远程解除**
- **锁定机制**：佩戴状态下 `canExitSOS = false`
- **充电联动**：SOS 中插入充电，显示充电页但保留 SOS 状态；拔电后自动回到 SOS 页
- **SOS 中禁止关机**
- **解除 SOS 后**：统一返回表情页（非背负异常页）

### 2.5 绑定状态管理
| 状态 | 权限限制 |
|------|----------|
| 已绑定 | 完整功能访问 |
| 未绑定 | 仅显示二维码页（**即使充电也只能看二维码**，异常页除外） |

- 绑定状态**随时可切换**（开机 / 关机 / 充电全状态通用）
- 切换时同步跳转：未绑定→二维码页（强制），已绑定→表情页 / 充电页
- **异常状态下禁止切换为"未绑定"**

### 2.6 充电模式
- 充电**前置校验**：仅"未检测"状态下可接入充电
- 充电时自动开机
- 充电时锁定：物理按键切页、背负状态切换、SOS 触发
- 关机状态插入充电：**未绑定**→二维码页；**已绑定**→充电页
- 开机状态插入充电：**未绑定**保持二维码页；**已绑定**→充电页
- 拔充电后恢复顺序：SOS > 异常 > 正常页面

### 2.7 设备异常页
- **支持类型**：网络异常 / 设备离线 / 电量过低 / **固件升级**
- **触发条件**：开机 && 非充电 && 非 SOS（**不限绑定状态**——异常优先于未绑定的二维码流程）
- **恢复行为**：
  - 网络异常 / 设备离线：解除后自动回原状态；页内停留 10 秒后息屏，唤醒时若未解除则继续回异常页
  - 电量过低：进入后 10 秒倒计时自动关机
  - **固件升级**：进入升级页显示进度条（0 → 100%，约 10 秒），完成后自动清除异常并返回对应页面
- **优先级**：SOS > 充电 > 异常 > 绑定状态 > 背负状态

## 3. 交互规则

### 3.1 物理按键操作
```
┌──────────────────────────────────────────────┐
│  操作类型   │  触发条件    │  功能             │
├──────────────────────────────────────────────┤
│  长按        │  按住 3 秒   │  开机 / 关机     │
│  短按        │  单击        │  切换页面        │
│  连击        │  500ms 内×3  │  触发 SOS        │
└──────────────────────────────────────────────┘
```

### 3.2 页面切换前置条件（全部满足才允许切页）
- 非充电
- 非 SOS
- 背负状态 = 未检测
- 已绑定

### 3.3 状态优先级
```
SOS 锁定 > 充电 > 设备异常 > 绑定状态 > 背负状态
```

## 5. 云端控制接口

### 5.1 远程解除 SOS
- **触发条件**：手表处于 SOS 状态
- **执行结果**：
  - 解除 SOS 锁定（`canExitSOS = true`）
  - 按优先级跳转：充电中 → 充电页；异常未解除 → 异常页；其余 → 表情页
  - 重置待机计时器

---

## 6. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 背负异常 | 停留背负页，不自动切页 |
| 待机唤醒 | 短按：异常未解除→异常页；否则按绑定状态回二维码/表情页 |
| 充电中按键 | 忽略操作（除 SOS 云端解除外） |
| 未绑定切页 | 强制返回二维码页（异常页除外；充电下也一样锁定） |
| 电量过低 | 异常页 10 秒倒计时后自动关机 |

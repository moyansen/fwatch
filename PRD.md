# 智能手表交互原型 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
智能手表高保真交互原型

### 1.2 产品定位
儿童/学生智能手表，主打安全定位、紧急求救、健康监测功能。

### 1.3 目标用户
- 6-15岁学生群体
- 家长（远程管理端）

---

## 2. 核心功能模块

### 2.1 电源管理
| 功能 | 说明 |
|------|------|
| 开机 | 长按物理按键5秒启动 |
| 关机 | 长按物理按键5秒关闭（佩戴状态下不可关机） |
| 待机 | 10秒无操作自动息屏 |
| 唤醒 | 短按按键从待机唤醒 |

### 2.2 页面导航
| 页面 | 内容 | 说明 |
|------|------|------|
| 待机屏 | 黑屏 | 省电模式 |
| Logo屏 | 品牌Logo + 加载动画 | 开机过渡页面 |
| 表情页 | 动态表情 😊 | 主界面/首页 |
| 信息页 | 身高、体重、学校、年级 | 用户基本信息 |
| 联系人页 | 紧急联系人信息 | 家长联系方式 |
| 步数页 | 今日步数统计 | 运动数据 |
| 二维码页 | 设备绑定二维码 + IMEI | 设备配对 |
| 背负页 | 背负状态显示 | 姿势监测 |
| SOS页 | 紧急求救信息 | 红色警示界面 |
| 充电页 | 充电动画 | 充电中显示 |

### 2.3 背负检测系统
| 状态 | 图标 | 说明 | 触发行为 |
|------|------|------|----------|
| 未检测 | ? | 灰色 | 10秒息屏 |
| 未背负 | ✕ | 灰色 | 正常显示 |
| 背负正常 | ✓ | 绿色 | 正常显示 |
| 背负异常 | ✗ | 红色 | 30秒倒计时后切页 |

### 2.4 SOS紧急求救
- **触发方式**：连按物理按键3次
- **退出方式**：
  - 非佩戴状态：连按3次退出
  - 佩戴状态：云端远程解除
- **锁定机制**：佩戴状态下SOS不可手动退出

### 2.5 激活状态管理
| 状态 | 权限限制 |
|------|----------|
| 已激活 | 完整功能访问 |
| 未激活 | 仅显示二维码页面（充电除外） |

### 2.6 充电模式
- 充电时自动开机
- 充电时锁定物理按键
- 充电时锁定背负状态切换
- 充电时显示充电页面
- 拔充电线后恢复未检测状态

## 3. 交互规则

### 3.1 物理按键操作
```
┌────────────────────────────────────────────┐
│  操作类型      │  触发条件      │  功能      │
├────────────────────────────────────────────┤
│  长按          │  按住5秒       │  开机/关机 │
│  短按          │  单击          │  切换页面  │
│  连击          │  500ms内3次    │  触发SOS   │
└────────────────────────────────────────────┘
```

### 3.2 页面切换规则
- 仅"未检测"状态下可切换页面
- 充电时禁止页面切换
- SOS激活时禁止页面切换
- 未激活设备锁定在二维码页

### 3.3 状态优先级
```
充电状态 > SOS状态 > 激活状态 > 背负状态
```

## 4. 技术规格

### 4.1 界面尺寸
- 表壳：320px × 320px（圆形）
- 屏幕：260px × 260px（圆形）
- 页面指示器：底部居中，6个圆点

### 4.2 动画效果
| 动画 | 应用场景 |
|------|----------|
| 旋转加载 | 开机Logo页 |
| 上下弹跳 | 表情页、充电页 |
| 脉冲闪烁 | SOS页警示图标 |
| 圆形进度 | 长按按键反馈 |

### 4.3 颜色规范
| 用途 | 颜色值 |
|------|--------|
| 主色调（科技蓝） | #0984e3 |
| 正常/成功（绿） | text-green-500 |
| 警告/异常（红） | text-red-500 |
| 未激活/禁用（灰） | text-gray-400/500 |

---

## 5. 云端控制接口

### 5.1 远程解除SOS
- **触发条件**：手表处于SOS状态
- **执行结果**：
  - 解除SOS锁定
  - 切换至表情页（或背负异常页）
  - 重置待机计时器

---

## 6. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 背负异常30秒无处理 | 自动返回表情页 |
| 待机唤醒 | 短按返回原页面 |
| 充电中按键操作 | 忽略操作 |
| 未激活切页 | 强制返回二维码页 |
